# **App Name**: UniTask

## Core Features:

- Autenticación de Usuario: Funcionalidad segura de registro e inicio de sesión de usuarios.
- Visualización del Horario: Muestra el horario del estudiante en una vista semanal clara. Permite agregar, editar o eliminar clases.
- Gestión de Tareas: Crea, edita y gestiona tareas académicas con fechas de vencimiento y niveles de prioridad.
- Recordatorios Inteligentes: Configura y recibe recordatorios automáticos para tareas y clases.
- Notificaciones Push: Envía notificaciones push oportunas para próximas clases, fechas límite y recordatorios.
- Gestión del Perfil de Usuario: Permite a los usuarios administrar su información de perfil, incluyendo nombre, correo electrónico, curso y contraseña.

## Style Guidelines:

- Color primario: Rosa suave (#FFB6C1) para evocar una sensación de calma y enfoque, beneficioso para el estudio y la organización.
- Color de fondo: Blanco hueso claro (#F8F8FF), ligeramente desaturado para mantener una interfaz suave y sin distracciones.
- Color de acento: Rosa pálido (#C21E56), para agregar interés visual sin ser abrumador; se utilizará para estados activos y acciones secundarias.
- Fuente del encabezado: 'PT Sans', sans-serif; también se utilizará para el cuerpo del texto.
- Utiliza iconos consistentes y minimalistas para facilitar la navegación y la claridad visual.
- Mantén un diseño limpio e intuitivo con secciones claras y mínimas distracciones.
- Implementa animaciones sutiles para transiciones y retroalimentación para mejorar la experiencia del usuario sin ser intrusivo.
